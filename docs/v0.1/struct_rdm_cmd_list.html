Detailed Description

A list of packed RDM Commands.

Two types of RPT messages contain an RdmCmdList: Request and Notification.

◆ partial

bool partial

This message contains a partial list.

This can be set when the library runs out of static memory in which to store RDM Commands and must deliver the partial list before continuing. The application should store the entries in the list but should not act on the list until another RdmCmdList is received with partial set to false.
