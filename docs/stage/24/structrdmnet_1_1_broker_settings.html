Detailed Description

A group of settings for Broker operation.

Field Documentation

◆ listen_addrs

std::set<etcpal::IpAddr> listen_addrs

A list of IP addresses representing network interfaces to listen on. If both this and listen_macs are empty, the broker will listen on all available interfaces. Otherwise listening will be restricted to the interfaces specified.

◆ listen_macs

std::set<etcpal::MacAddr> listen_macs

A list of MAC addresses representing network interfaces to listen on. If both this and listen_addrs are empty, the broker will listen on all available interfaces. Otherwise listening will be restricted to the interfaces specified.

◆ listen_port

uint16_t listen_port {0}

The port on which this broker should listen for incoming connections (and advertise via DNS). 0 means use an ephemeral port.

◆ max_reject_connections

unsigned int max_reject_connections {1000}

If you reach the number of max connections, this number of tcp-level connections are still supported to reject the connection request.
