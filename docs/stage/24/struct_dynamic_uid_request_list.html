Field Documentation

◆ more_coming

bool more_coming

This message contains a partial list.

This can be set when the library runs out of static memory in which to store DynamicUidRequestListEntrys and must deliver the partial list before continuing. The application should store the entries in the list but should not act on the list until another DynamicUidRequestList is received with partial set to false.
