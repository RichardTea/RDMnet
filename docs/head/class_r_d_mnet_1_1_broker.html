Detailed Description

Defines an instance of RDMnet Broker functionality.

After instantiatiation, call Startup() to start Broker services on a set of network interfaces. Starts some threads (defined in broker/threads.h) to handle messages and connections. Periodically call Tick() to handle some cleanup and housekeeping. Call Shutdown() at exit, when Broker services are no longer needed, or when a setting has changed.

Start all Broker functionality and threads.

If listen_addrs is empty, this returns false. Otherwise, the broker uses the address fields to set up the listening sockets. If the listen_port is 0 and their is only one listen_addr, an ephemeral port is chosen. If there are more listen_addrs, listen_port must not be 0.

Parameters
[in]settingsSettings for the Broker to use for this session.
[in]listen_portPort for the Broker to listen on.
[in]listen_addrsAddresses of network interfaces for the Broker to listen on.
Returns
true (started Broker successfully) or false (an error occurred starting Broker).
