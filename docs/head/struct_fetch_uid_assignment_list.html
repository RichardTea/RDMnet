Detailed Description

A list of Dynamic UIDs for which the currently assigned Responder IDs (RIDs) are being requested.

Field Documentation

FetchUidAssignmentListEntry* assignment_list

The head of a linked list of Dynamic UIDs for which the currently assigned RIDs are being requested.

bool more_coming

This message contains a partial list.

This can be set when the library runs out of static memory in which to store FetchUidAssignmentListEntrys and must deliver the partial list before continuing. The application should store the entries in the list but should not act on the list until another DynamicUidRequestList is received with partial set to false.
