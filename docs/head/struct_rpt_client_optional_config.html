Detailed Description

The optional values contained in an RPT Client configuration.

These values have defaults that can be initialized using the appropriate initialization macros.

Field Documentation

const char* search_domain

The client's configured search domain for discovery.

RdmUid uid

The client's UID.

If the client has a static UID, fill in the values normally. If a dynamic UID is desired, assign using RPT_CLIENT_DYNAMIC_UID(manu_id), passing your ESTA manufacturer ID. All RDMnet components are required to have a valid ESTA manufacturer ID.
