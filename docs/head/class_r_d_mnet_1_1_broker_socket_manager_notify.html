bool  graceful 
)
pure virtual

A socket was closed remotely.

The socket is no longer valid after this callback finishes. Do not call BrokerSocketManager::RemoveSocket() or any other API function from this callback as it is unnecessary and may cause a deadlock.

Parameters
[in]conn_handleThe RDMnet connection handle for which the socket was closed.
[in]gracefulWhether the TCP connection was closed gracefully.
