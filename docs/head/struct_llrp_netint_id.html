Detailed Description

A set of identifying information for a network interface, for LLRP's purposes.

All messages in LLRP are sent over multicast. This means that interface IP addresses don't matter and that for the purposes of LLRP, the primary key for a network interface should simply be a combination of the interface index and the IP protocol used.
