void rdmnet_llrp_target_update_connection_state ( llrp_target_t  handle,
bool  connected_to_broker 
)

Update the Broker connection state of an LLRP target.

If an LLRP target is associated with an RPT client, this should be called each time the client connects or disconnects from a broker. Controllers are considered not connected when they are not connected to any broker. This affects whether the LLRP target responds to filtered LLRP probe requests.

Parameters
[in]handleHandle to LLRP target for which to update the connection state.
[in]connected_to_brokerWhether the LLRP target is currently connected to a broker.
