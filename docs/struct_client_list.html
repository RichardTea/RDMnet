Detailed Description

A structure that represents a list of Client Entries.

Represents the data for multiple Broker Protocol messages: Connected Client List, Client Incremental Addition, Client Incremental Deletion, and Client Entry Change.

◆ partial

bool partial

This message contains a partial list.

This can be set when the library runs out of static memory in which to store Client Entries and must deliver the partial list before continuing. The application should store the entries in the list but should not act on the list until another ClientList is received with partial set to false.
